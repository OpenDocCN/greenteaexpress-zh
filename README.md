## 下载

### Docker

```
docker pull apachecn0/greenteaexpress-zh
docker run -tid -p <port>:80 apachecn0/greenteaexpress-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g greenteaexpress-zh
greenteaexpress-zh <port>
# 访问 http://localhost:{port} 查看文档
```
