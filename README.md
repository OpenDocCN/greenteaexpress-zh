## 下载

### Docker

```
docker pull apachecn0/greenteapress-zh
docker run -tid -p <port>:80 apachecn0/greenteapress-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g greenteapress-zh
greenteapress-zh <port>
# 访问 http://localhost:{port} 查看文档
```
